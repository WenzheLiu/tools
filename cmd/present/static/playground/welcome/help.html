Hello, Welcome to WebPPT Playground!

You can learn WebPPT here and play it with in-time render.

  • The left area is to edit the slide file
  • The right area is to show the rendered slide
  • After edit the text, you need to click Run button at the header to refresh
  • A default welcome page will be shown at first time. It has in-line comments to help you learn how to write WebPPT. If you want to retrieve the welcome page back, you can click Welcome button below or visit here.
  • The field besides Run button is Resource URL. You can specify where to find the resources that used in your slide.
Now you can start your WebPPT tour. Enjoy!
